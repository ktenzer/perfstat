Summary

Hosts are what the PerfStat application is built around. A host is any device being monitored by the PerfStat application. A host could be a server, network device, or anything with an IP Address.

Add a Host

1) Select "App Configs->Host Config" within the PerfStat application

2) Under the "Add Host" section enter the Host Name, IP Address, select the OS Type from a drop down list and select "Enter" to add host

<PICTURE>

Modify Host

1) Select "App Configs->Host Config" within the PerfStat application

2) Under the "Manage Hosts" section select "Config" to configure host

3) Under the "Modify Host" section modify the Host Name and/or IP Address and select "Enter" to modify host

<PICTURE>

Delete Host

Data is associated with a host in the PerfStat application. If you delete a host from the PerfStat application all it's status and performance data will also be deleted.

1) Select "App Configs->Host Config" within the PerfStat application

2) Under the "Manage Hosts" section select "Delete" to delete host, you will be prompted before host is deleted
