Summary

The perfstat cli tools are used to configure and pull data from the perfstat application. All tools are located in the /perfhome/bin/tools directory.

CLI Tools

perfconfig - used to configure the perfstat server and client.

/perfhome/bin/tools/perfconfig [-list] [-root]

/perfhome/bin/tools/perfconfig [-server] [-root]

/perfhome/bin/tools/perfconfig [-client] [-root]

/perfhome/bin/tools/perfconfig [-global] [-root]

perfreport - used to extract performance data for a specified host->service->metric during a specified time period. It will print a timestamp, performance data, and the metric name.

/perfhome/bin/tools/perfreport.pl [-h <hostname>] [-s <service>] [-m <metric1,metric2,...>|<all>] [-begin <now>|<DATE><TIME>] [-end <now>|<DATE><TIME>] [-dump]

-h - hostname

-s - service

-m - metric(s) or all

-begin - start date and time format: xx/xx/xxxx xx:xx

-end - end date and time format: xx/xx/xxxx xx:xx

-dump - prints only the performance data, no label and no timestamp

IE: /perfhome/bin/tools/perfreport.pl -h myhost.mydomain.com -s cpu -m all -begin 1/1/2002 -end now

/perfhome/bin/tools/perfreport.pl [-h <hostname>] [-list <services|serviceName>]
