Summary

This section contains information for installing the Perfstat client on redhat linux hosts. The perfstat client for redhat comes packaged as an rpm. By default the rpm will install into /opt/PERFclt.

RPM Installation

1) Copy the perfstat_vX.XX-rhatXXXX.rpm to the /tmp directory.

2) Run the following command to install the rpm:

rpm -ivh /tmp/perfstat_vX.XX-rhatXXXX.rpm

Perfstat Client Installation

3) Create perfstat user by running the following command:

useradd -d /opt/PERFCLT psuser

4) Create perfstat group by running the following command:

groupadd psgroup

5) Add perfstat user as a member of the perfstat group by running the following command:

usermod -g psgroup psuser

6) Run the following command to change directory into the perfstat client install directory:

cd /opt/PERFclt/install

7) Run the following command to install and configure the perfstat client:

./perfinstall.sh /opt/PERFclt

8) Enter psuser as the perfstat user.

9) Enter psgroup as the perfstat group.

10) Enter "Y" for yes to allow the installation program to change permissions of the /opt/PERFclt directory.

11) Enter "Y" for yes to allow the installation program to start perfstat automatically upon boot.

Perfstat Global Configuration

12) Enter path to perfhome directory /opt/PERFclt

13) Enter psuser as the perfstat user.

14) Enter psgroup as the perfstat group.

15) Enter 0 for debug level.

16) Enter "Y" for yes to save global settings.

Perfstat Client Configuration

17) Enter the IP address of the perfstat server. By default it will grab the IP address of the current host.

18) Enter the port for the perfstat server.

19) Enter perfstat client services. You should except the default unless you have special requirements.

20) Enter perfstat client run interval. You should except the default unless you need more or less granular data.

21) Enter path to ps command. The configuration program will try and find the path automatically buy you should double check.

22) Enter path to vmstat command. The configuration program will try and find the path automatically buy you should double check.

23) Enter path to iostat command. The configuration program will try and find the path automatically buy you should double check.

24) Enter path to netstat command. The configuration program will try and find the path automatically buy you should double check.

25) Enter path to df command. The configuration program will try and find the path automatically buy you should double check.

26) Enter "Y" for yes to save client settings.

If something changes later on, you can always re-run the installation program or the configuration program manualy. The configuration program is located in /perfhome/bin/tools/perfconfig.
