Summary

This section contains information for installing the Perfstat server on sun solaris hosts. The perfstat server for solaris comes as a package. By default the package will install into /opt/PERFsvr.

Package Installation

1) Copy the perfstat_vX.XX-solXXXX.pkg to the /tmp directory.

2) Run the following command to install the rpm:

pkgadd -d /tmp/perfstat_vX.XX-solXXXX.rpm

Perfstat Server Installation

3) Create perfstat user by running the following command:

useradd -d /opt/PERFsvr psuser

4) Create perfstat group by running the following command:

groupadd psgroup

5) Add perfstat user as a member of the perfstat group by running the following command:

usermod -g psgroup psuser

6) Run the following command to change directory into the perfstat client install directory:

cd /opt/PERFsvr/install

7) Run the following command to install and configure the perfstat client:

./perfinstall.sh /opt/PERFsvr

8) Enter psuser as the perfstat user.

9) Enter psgroup as the perfstat group.

10) Enter "Y" for yes to allow the installation program to change permissions of the /opt/PERFsvr directory.

11) Enter "Y" for yes to allow the installation program to start perfstat automatically upon boot.

Perfstat Global Configuration

12) Enter path to perfhome directory /opt/PERFsvr

13) Enter psuser as the perfstat user.

14) Enter psgroup as the perfstat group.

15) Enter 0 for debug level.

16) Enter "Y" for yes to save global settings.

Perfstat Server Configuration

17) Enter IP address of host you are configuring as perfstat server. By default it will grab the IP address of the current host.

18) Enter port you would like perfstat server daemon to listen on. For security you should always use a port greater than 1024. The default is 2002.

19) Enter max bytes for an individual perfstat server connection. The default is 8KB.

20) Enter perfstat server ping interval. Interval perfstat will use to ping clients. The default is 5 minutes.

21) Enter perfstat server status interval. Interval perfstat will use for checking client status. The default is 30 minutes.

22) Enter "Y|N" depending on if you would like to enable email alerts. The default is "N"

22a) Enter "Y|N" depending on if you would like to enable email alerts for all event status changes.

22b) Enter "Y|N" depending on if you would like to enable email alerts for crit status changes.

22c) Enter "Y|N" depending on if you would like to enable email alerts for warn status changes.

22d) Enter "Y|N" depending on if you would like to enable email alerts for nostatus status changes.

22e) Enter IP address of SMTP server. The SMTP server can be a mail relay.

23) Enter "Y|N" depending on if you would like to enable event logging. The default is "Y"

23a) Enter max event log size in number of lines. The default is 100.

Perfstat Client Configuration

24) Enter the IP address of the perfstat server. By default it will grab the IP address of the current host.

25) Enter the port for the perfstat server.

26) Enter perfstat client services. You should except the default unless you have special requirements.

27) Enter perfstat client run interval. You should except the default unless you need more or less granular data.

28) Enter path to ps command. The configuration program will try and find the path automatically buy you should double check.

29) Enter path to vmstat command. The configuration program will try and find the path automatically buy you should double check.

30) Enter path to iostat command. The configuration program will try and find the path automatically buy you should double check.

31) Enter path to netstat command. The configuration program will try and find the path automatically buy you should double check.

32) Enter path to df command. The configuration program will try and find the path automatically buy you should double check.

33) Enter path to swap command. The configuration program will try and find the path automatically buy you should double check.

34) Enter path to prtconf command. The configuration program will try and find the path automatically buy you should double check.

35) Enter path to ifconfig command. The configuration program will try and find the path automatically buy you should double check.

36) Enter path to kstat command. The configuration program will try and find the path automatically buy you should double check.

37) Enter "Y" for yes to save client settings.

If something changes later on, you can always re-run the installation program or the configuration program manualy. The configuration program is located in /perfhome/bin/tools/perfconfig.
