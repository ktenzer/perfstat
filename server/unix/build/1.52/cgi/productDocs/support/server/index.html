Summary

The PerfStat server application supports the Redhat Linux and Sun Solaris operating systems. However, since the application can be run as non-compiled code, it is possible to port the PerfStat client application to virtually any platform that supports perl 5.8.

Redhat Linux Minimum Requirements

ES3/ES 4

Pentium IV 2.4GHZ

1GB RAM

500MB Free Disk Space + 10MB for every client

ActiveState Perl 5.8

Apache Web Server

100 Clients per CPU

Solaris (SPARC) Minimum Requirements

Solaris 7/8/9

Ultra Sparc III 1GHZ

1GB RAM

500MB Free Disk Space + 10MB for every client

ActiveState Perl 5.8

Apache Web Server

100 Clients per CPU
