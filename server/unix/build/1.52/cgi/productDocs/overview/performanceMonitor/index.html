Summary

Performance monitor is a custom interface for creating and displaying both host and host group graphs. You can create three types of graphs: pie, bar, and line. Note: currently line graphs are not available for host group graphs.

Host Graphs - Can be pie, bar, or line graphs that reflect the data of a single host.

Host Group Graphs - Can be pie or bar graphs that reflect the data of all hosts in a host group.

Selecting Graphs

Performance monitor lets you display two type of graphs: host group and host graphs.

To select a host group graph ensure the correct host group is chosen and Left Click the appropriate service->graph

<Picture>

To select a host graph you must first select the appropriate host group where that host resides and the Left Click "Drill Down to Host Graphs". Next Left Click the appropriate service->graph

<Picture>

Selecting Options

When selecting graph(s) you have the following options: open all, close all, select all, and remove all.

Open All - Will open all the services for a given host group or host.

Close All - Will close all the services for a given host group or host.

Select Aall - Will select all graphs for a given host group or host.

Remove All - Will remove all graphs for a given host group or host.

Display Graph(s)

Once one or more host group or host graphs are selected simply Left Click create to display the graph(s).

Display Options

When displaying graph(s) you have the following options: layout, graph size, and custom size.

Layout - Allows you to span your graphs accross multiple columns. You can choose between one to four columns

Graph Size - Creates graph(s) based on the following sizes: small, medium, and large.

Custom Size - Allows you to create graph(s) with a custom size. You can choose between 50 to 300 percent. Note: custom sizes are based on the medium sized graphs.
