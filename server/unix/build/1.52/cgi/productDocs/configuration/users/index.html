Summary

The PerfStat application is a multi-user environment. All users can create/modify/delete hosts groups, reports, and control certain user specific enviornmentals. Only admin users and the perfstat user can add/modify/delete hosts, other users, and control system environmentals.

Add User

Only an admin user or the perfstat user can add other users. Users created by an admin or the perfstat user will only see hosts added to the PerfStat application by that admin or the perfstat user.

1) Select "App Configs->User Config" within the PerfStat application

2) Under the "Add User" section select a User Name and password, select "Enter" to add that user

Modify User

Only an admin user or the perfstat user can modify other users. Users created by an admin or the perfstat user will only see hosts added to the PerfStat application by that admin or the perfstat user.

1) Select "App Configs->User Config" within the PerfStat application

2) Under the "Add User" section select "Config" to modify that user

3) Under the "Modify User" section enter a new password and select "Enter" to modify that users password

4) Under the "Preference" section set "Show all Hosts" to true or false to enable or disable the "All Hosts" Host Group from displaying under Shared HostGroups

<PICTURE>

Delete User

When you delete a user from the PerfStat application all the user settings, host groups, and reports will also be deleted.

1) Select "App Configs->User Config" within the PerfStat application

2) Under the "Manage Users" section select "Delete" to delete that user

Add Admin User

1) Select "App Configs->User Config" within the PerfStat application

2) Under the "Add Admin" section select a User Name and password, select "Enter" to add that Admin user

Modify Admin User

Only the perfstat user can add admin users to the PerfStat application.

1) Select "App Configs->User Config" within the PerfStat application

2) Under the "Manage Admin" section section select "Config" to modify that Admin user

3) Under the "Modify User" section enter a new password and select "Enter" to modify that users password

4) Under the "Preference" section set "Show all Hosts" to true or false to enable or disable the "All Hosts" Host Group from displaying under Shared HostGroups

1) Select "App Configs->User Config" within the PerfStat application
